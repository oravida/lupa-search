## REQUISITOS TÉCNICOS DOS NOVOS BLOCOS

- Todos os SVGs do mapa do Brasil devem ser criados inline no HTML com paths simplificados mas reconhecíveis — não usar imagens externas
- Todos os gráficos devem ser construídos em CSS puro e/ou SVG inline — sem bibliotecas externas como Chart.js
- Animações devem usar o IntersectionObserver já implementado no site — apenas acrescentar as classes necessárias
- Os novos blocos devem ser totalmente responsivos: em mobile, os elementos lado a lado devem empilhar verticalmente
- Manter a tipografia já definida: `Playfair Display` para títulos e `Source Sans 3` para corpo
- Tooltips dos mapas devem funcionar tanto em hover (desktop) quanto em tap (mobile)
- Nenhuma alteração nas seções, cores, textos ou estrutura já existentes no site
- Os novos blocos devem ter padding/espaçamento consistente com o restante do site
