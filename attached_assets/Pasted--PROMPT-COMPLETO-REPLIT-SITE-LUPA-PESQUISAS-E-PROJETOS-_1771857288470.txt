## REQUISITOS TÉCNICOS GERAIS

- HTML5 semântico e válido
- CSS3 com variáveis CSS (`:root`) para toda a paleta de cores
- JavaScript puro (sem frameworks) para: menu hambúrguer, scroll suave para âncoras, validação do formulário, fade-in das seções ao entrar no viewport (IntersectionObserver)
- Totalmente responsivo: mobile-first. Breakpoints em 768px e 1024px
- Google Fonts: importar `Playfair Display` e `Source Sans 3`
- Todos os botões com transição suave de 0.3s no hover
- Sem bibliotecas externas pesadas (sem Bootstrap, sem jQuery)
- Carregamento rápido: imagens com lazy loading, SVGs inline onde possível
- Favicon simples com a letra "L" em laranja
- Meta tags de SEO básicas em ambas as páginas (title, description, viewport)
- Placeholder de logo: SVG circular com fundo laranja `#faa749`, anel turquesa `#55c5d0`, anel cinza `#58595b` e texto "LUPA" em grafite — fiel à logo real

## OBSERVAÇÕES FINAIS PARA O REPLIT

- Criar os dois arquivos: `index.html` e `sobre.html`
- Todo CSS pode estar em arquivo separado `style.css` ou dentro de `<style>` no HTML
- Todo JavaScript pode estar em arquivo separado `script.js` ou no final do HTML antes de `</body>`
- NÃO usar animações de texto em movimento, efeito typewriter ou scroll infinito de palavras
- Manter coerência visual total entre as duas páginas
- Testar responsividade em mobile antes de finalizar
